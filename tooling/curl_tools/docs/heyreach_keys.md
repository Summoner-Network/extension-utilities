# How to get API keys

Go to the [HeyReach login page](https://app.heyreach.io/account/login), enter the web app, and follow the following steps:

1. Go to the [Dashboard screen](https://app.heyreach.io/app/dashboard)
2. Navigate to [Integrations](https://app.heyreach.io/app/integrations) from left menu.
2. On the "Integrations" screen, click "Get API key" in the "HeyReach API" widget.
3. On the [HeyReach API key screen](https://app.heyreach.io/app/integrations/public-api/api), copy the generated API key (or click on "New API key" for a new one)
6. Create `.env` and add your "API key" as follows:
    ```sh
    HEYREACH_API_KEY=you-api-key-here
    ```
